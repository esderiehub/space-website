# COSMOS  
A dark-themed space and astronomy platform that brings the universe to your screen.  

## About 
COSMOS is a visually immersive web experience dedicated to space exploration,  
planets, and the solar system. Built with a minimalist dark design and smooth  
animations to match the depth of the universe.  

## Features  
- Animated preloader screen  
- Scroll-based reveal animations  
- Counter animations  
- Information on planets and space exploration  
- Fully responsive and minimalist UI
